🔧 JSON 格式化工具

用途：美化、压缩、校验 JSON 数据。

使用技巧：

  • 粘贴 JSON 后点击"格式化"按钮，自动美化
  • 点击"压缩"可移除空格，减小文件大小
  • 输入无效 JSON 会自动定位错误位置

🔐 Base64 编解码

用途：文本/图片的 Base64 编码和解码。

使用技巧：

  • 支持 URL 安全编码（适用于 URL 参数）
  • 可拖拽上传图片，自动转换为 Base64
  • 解码时可选择下载为文件
