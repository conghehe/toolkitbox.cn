一、什么是SEO？

SEO（Search Engine Optimization，搜索引擎优化）是通过优化网站，提高在搜索引擎自然搜索结果中排名的过程。与付费广告不同，SEO带来的是免费、持续、高质量的流量。

💡 为什么SEO重要？
- Google 首页前3位获得 75% 的点击
- 自然搜索流量的转化率比付费广告高 8.5 倍
- 长期投资回报率（ROI）远超其他营销渠道

二、关键词研究

1. 如何选择关键词

  • 搜索量：使用 Google Keyword Planner、Ahrefs 等工具
  • 竞争度：新网站优先选择长尾关键词（3-5个词）
  • 商业意图：导航词（品牌）> 交易词（购买）> 信息词（学习）

2. 关键词布局

  1. 标题标签（Title Tag）：包含主关键词，长度 50-60 字符
  2. Meta Description：包含主关键词，长度 150-160 字符，吸引点击
  3. H1-H6 标签：主关键词在 H1，相关词在 H2-H3
  4. 正文内容：关键词密度 1-2%，自然出现

三、内容优化

1. 内容质量标准

  • 原创性：避免抄袭，提供独特见解
  • 深度：覆盖主题的所有方面，内容长度 1500+ 字
  • 可读性：短段落、小标题、列表、图片
  • 时效性：定期更新内容，保持新鲜度

2. 用户体验优化

  • 页面加载速度 < 2秒（使用 Google PageSpeed Insights 检测）
  • 移动端适配（Mobile-Friendly）
  • 清晰的导航和内链结构
  • 降低跳出率（提供相关内容推荐）

四、技术SEO

1. 网站结构优化

  • 提交 XML Sitemap 到 Google Search Console
  • 使用 Robots.txt 控制爬虫访问
  • 设置 301 重定向（避免重复内容）
  • 使用 HTTPS（SSL 证书）

2. 页面技术优化

  • 图片优化：压缩 + Alt 属性 + 描述性文件名
  • 结构化数据（Schema Markup）：帮助搜索引擎理解内容
  • 规范标签（Canonical Tag）：避免重复内容问题
  • 页面速度优化：压缩 CSS/JS、启用 Gzip、使用 CDN

五、外链建设

外链（Backlinks）是 Google 排名最重要的因素之一。高质量外链 = 其他权威网站链接到你的网站。

1. 外链获取方法

  • 内容营销：创建值得链接的高质量内容（研究、数据、工具）
  • 客座博客：在行业网站发布文章，附带链接
  • 资源页链接：寻找行业资源页，申请添加你的网站
  • broken Link Building：找到失效外链，建议替换为你的内容

2. 外链质量判断

  • 域名权重（Domain Authority, DA）：DA 30+ 为佳
  • 相关性：来自同行业网站的链接更有价值
  • 锚文本自然：避免过度优化（如所有链接都用精确匹配关键词）
  • 多样性：来自不同域名，而非同一个网站的大量链接

  • Google PageSpeed Insights：免费，检测页面速度
