使用说明

  • 上传图片：点击上传区域或拖拽图片文件到页面中，支持批量上传多张图片
  • 选择格式：在下拉菜单中选择目标输出格式（PNG / JPG / WebP / AVIF）
  • 调节质量：拖动质量滑块调整输出画质（仅对有损格式生效，PNG为无损忽略质量设置）
  • 开始转换：点击"全部转换"按钮，系统将使用 Canvas API 进行格式转换
  • 下载图片：点击单张图片下的"下载"按钮，或使用"全部下载"批量保存

格式转换建议

  • 照片 → WebP：摄影作品和网页展示照片推荐转换为 WebP，体积减小 30% 以上
  • 截图 → PNG：UI 截图和文字图片推荐保留 PNG 格式，保证清晰度
  • 社交分享 → JPG：微博、微信等社交平台分享推荐 JPG 格式，兼容性最好
  • 现代网站 → AVIF：追求极致性能的网站可转换为 AVIF 格式，体积最小

常见问题

Q: 转换会降低图片质量吗？

A: 转换为有损格式（JPG/WebP/AVIF）时可通过质量滑块控制画质。转换为无损格式（PNG）时不会损失质量。建议质量设置在 80%-90% 之间，肉眼几乎无法察觉差异。

Q: 支持哪些图片格式输入？

A: 支持所有主流浏览器可读取的图片格式：PNG、JPG/JPEG、WebP、AVIF、BMP、GIF、SVG、ICO 等。

Q: 上传的图片会上传到服务器吗？

A: 不会。所有图片处理均在您的浏览器本地完成，使用 Canvas API 进行格式转换，图片数据不会离开您的设备。

Q: 可以同时转换多张图片吗？

A: 可以。支持批量上传和批量转换，每张图片独立处理，互不影响。

Q: JPG 转 PNG 文件会变大吗？

A: 通常会变大。PNG 是无损格式，JPG 是有损压缩格式，将 JPG 转为 PNG 时文件体积可能增大。如果不需要透明背景，建议保持 JPG 格式或转为 WebP。

Q: AVIF 格式兼容性如何？

A: AVIF 是较新的图片格式，Chrome 85+、Firefox 93+、Safari 16.1+ 均已支持，覆盖率约 93%。对于需要兼容旧浏览器的场景，建议使用 WebP 作为替代。
