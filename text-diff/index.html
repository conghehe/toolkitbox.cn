在线文本差异对比工具 — 免费 Diff Checker

本工具提供实时在线文本差异对比功能，支持逐行对比、差异高亮、并排视图和行内统一视图。适用于代码审查、文档版本对比、配置文件变更检测等场景。所有处理均在浏览器本地完成，您的文本数据不会上传到任何服务器，确保数据安全。

核心功能

  • 双视图切换：并排视图（Side-by-Side）直观展示左右差异；行内视图（Unified View）像 git diff 一样紧凑展示变更。
  • 差异高亮：新增行以绿色背景标记，删除行以红色背景标记，修改行以黄色背景标记。
  • 行号显示：精确显示每行在原文和修改后文本中的位置。
  • 本地处理：基于 JavaScript LCS（最长公共子序列）算法，纯浏览器端运行，无需网络请求。
  • 复制/下载：一键复制 diff 结果为纯文本，或下载为 .diff 文件。
  • 深色主题：支持浅色/深色两套主题，保护眼睛。

适用场景

场景说明
代码审查对比代码修改前后的差异，快速定位变更点和潜在问题。
文档版本对比比较两版合同、协议、技术文档的文字差异。
配置文件变更检测 JSON/YAML/TOML 等配置文件的字段增删改。
SQL 脚本审核对比 DDL/DML 变更，确保数据库迁移脚本正确。
日志分析比较两次日志输出的差异，定位异常变化。
翻译校对对比原文与翻译文本，检查遗漏或错误。

使用说明

1. 在左侧文本框粘贴原始文本（旧版本）
2. 在右侧文本框粘贴修改后文本（新版本）
3. 点击「对比差异」按钮或使用快捷键 Ctrl+Enter
4. 查看差异结果，可切换并排/行内视图
5. 点击「复制结果」导出纯文本 diff，或「下载 Diff」保存文件

常见问题

Q: 支持多大的文本？
A: 建议单侧文本不超过 5000 行或 500KB。超大文本可能影响对比速度，建议分段比较。

Q: 数据会上传吗？
A: 不会。所有文本处理和对比计算完全在您的浏览器中完成，数据不会离开您的设备。

Q: 与 git diff 有什么区别？
A: 本工具提供可视化的差异对比界面，操作简单直观。git diff 更适合命令行和版本控制场景，本工具更适合临时快速对比和团队协作演示。

Q: 支持哪些文本编码？
A: 支持 UTF-8 编码的所有文本，包括中文、英文、代码、JSON 等。
