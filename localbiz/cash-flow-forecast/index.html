📐 计算公式与说明

现金流预测方法论

1. 月度净现金流 = 月收入 − (固定成本 + 变动成本 + 一次性支出)
当启用季节性调整时：月收入 = 基础月收入 × 该月季节系数

2. 累计现金流 = 上月累计现金流 + 本月净现金流
累计现金流反映企业资金储备的变化趋势。若连续多月为负，说明在消耗储备金。

3. 现金流健康评分 (0-100) = 基于以下维度加权计算：
  • 净现金流为正的月份占比 (权重30%)
  • 累计现金流是否始终为正 (权重25%)
  • 年度净现金流/年收入比率 (权重20%)
  • 最大单月亏损占月收入的比例 (权重15%)
  • 最低月现金储备是否足够覆盖3个月支出 (权重10%)
评分 >80 = 健康 | 60-80 = 关注 | 40-60 = 风险 | <40 = 严重

4. 烧钱月数 = 当累计现金流逐步下降时，从当前储备耗尽所需月数。
数据来源：月收入/成本参考值基于中国小微企业调查数据(Ant Group 2025年小微经营报告)、各行业平均毛利率(中国连锁经营协会2025)、人工成本(各地最低工资标准汇总)。
