数据安全说明

本工具所有数据生成完全在您的浏览器中完成，使用 JavaScript 随机数引擎。不会将任何数据传输到服务器，也不会存储或记录您的使用行为。生成的信用卡号、身份证号等均为符合格式规则的虚拟数据，不可用于真实交易或身份验证。

常见问题

Q: 生成的信用卡号是真实的吗？
A: 不是。生成的卡号仅通过 Luhn 校验算法，格式上与真实卡号一致，但无法通过支付网关的实际验证。仅供开发测试使用。

Q: 可以一次生成超过 100 条吗？
A: 默认限制 100 条以确保页面性能。如需更大批量，可以使用批量 x100 快速生成，或通过 JSON 导出后在脚本中循环拼接。

Q: 中文姓名是否使用了真实数据？
A: 姓名基于公开的常见姓氏和名字用字分布随机组合生成，不关联任何真实个人信息。

Q: 支持哪些导出格式？
A: 支持 JSON 和 CSV 两种格式。JSON 适合程序读取，CSV 适合 Excel / Google Sheets 打开。
