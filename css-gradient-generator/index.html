什么是 CSS 渐变？

CSS 渐变（Gradient）是一种CSS背景图像类型，由两种或多种颜色之间的渐进过渡组成。渐变可以作为 background-image 属性的值，在现代Web设计中广泛用于创建按钮、背景、边框、叠加层等视觉效果。CSS渐变有以下几种类型：

  • 线性渐变（linear-gradient）：颜色沿直线方向过渡，可指定角度如 90deg、135deg，或使用关键字 to right、to bottom left 等。
  • 径向渐变（radial-gradient）：颜色从一个中心点向外辐射过渡，支持 circle/ellipse 形状和位置控制。
  • 锥形渐变（conic-gradient）：颜色围绕中心点旋转过渡，适合饼图、色轮等效果。
  • 重复渐变：repeating-linear-gradient / repeating-radial-gradient 可创建条纹、波纹等重复图案。

常见问题

Q: 渐变和背景图片哪个性能更好？
A: CSS渐变由浏览器原生渲染，不产生HTTP请求，性能优于背景图片。尤其适合作为页面背景、按钮等高频元素。

Q: 渐变的浏览器兼容性如何？
A: linear-gradient 和 radial-gradient 在所有现代浏览器中都得到良好支持（Chrome 26+, Firefox 16+, Safari 6.1+, Edge 12+）。conic-gradient 支持 Chrome 69+, Firefox 83+, Safari 12.1+。

Q: 如何在文字上使用渐变？
A: 可以使用 background-clip: text 配合 -webkit-text-fill-color: transparent 实现文字渐变效果。
