什么是 Markdown？

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创建。它使用简单的纯文本格式语法，可以轻松转换为结构化的 HTML 文档。Markdown 被广泛应用于 GitHub、Stack Overflow、Reddit、Notion 等平台，是现代开发者和写作者必备的技能之一。

Markdown 编辑器常见问题

Q: Markdown 编辑器和富文本编辑器有什么区别？
A: Markdown 编辑器用纯文本语法标记格式，文件体积小、跨平台兼容性好、可版本控制。富文本编辑器所见即所得，但文件格式不通用。

Q: 我的数据安全吗？
A: 完全安全。本工具是纯前端应用，所有编辑和渲染操作都在你的浏览器中完成，数据不会上传到任何服务器。

Q: 支持哪些 Markdown 语法？
A: 支持 GitHub Flavored Markdown (GFM) 规范，包括标题、加粗、斜体、删除线、链接、图片、列表、引用、代码块、表格、任务列表、分割线等完整语法。
